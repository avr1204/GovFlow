# GovFlow AI Assistant

## Overview
GovFlow is a web application that uses AI to help users interact with government-related services more easily. It provides simple responses to user questions and helps reduce confusion when dealing with forms or processes.

---

## Features
- AI chatbot for answering questions  
- Simple and clean web interface  
- Real-time responses  
- Deployed online using Google Cloud Run  

---

## Tech Stack
- Java (Spring Boot)  
- HTML, CSS, JavaScript  
- Google Cloud Run  
- Docker  

---

## AI Usage
AI was used to:
- assist in coding and debugging  
- generate responses for user input  

Tools used:
- Gemini

---

## Live Website
https://govflow-507818934198.asia-southeast1.run.app

---

## How It Works
User enters a question → frontend sends request → backend processes → AI generates response → result shown to user
